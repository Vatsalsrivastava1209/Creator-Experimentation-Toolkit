# Post Evidence Folder

Store screenshots of the real X posts here after the experiment is run.

Recommended naming:

```text
tweet_001_variant_A.png
tweet_002_variant_B.png
...
tweet_034_variant_B.png
```

Each screenshot should make it possible to verify:

- the post exists
- the hook text matches `tweet_schedule.csv`
- the posted date/time is plausible
- the public engagement metrics match `data/real_experiment_results.csv`
- the tweet URL or tweet ID can be connected back to the result row

Do not add screenshots for synthetic sample rows. This folder is meant for real
experiment evidence only.
